# Meshaal Alharbi - Portfolio Website (Assignment 2)

This project is a personal portfolio website developed for SWE 363 Assignment 2.  
It is an improved version of Assignment 1 with added interactivity and better user experience.

## Features
- About section with a dynamic greeting message
- Projects section with:
  - Live search functionality
  - Category filter buttons (All, Web, Java)
  - Show/Hide project details
- Contact form with on-page success message
- Dark/Light theme toggle using localStorage
- Responsive design for desktop, tablet, and mobile devices

## How to Run Locally
1. Download or clone the repository  
2. Open the project folder in VS Code  
3. Open `index.html` in your browser  
   or use the Live Server extension
