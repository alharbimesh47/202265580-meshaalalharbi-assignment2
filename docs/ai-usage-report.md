# AI Usage Report

## Tools Used and Use Cases
During this assignment, I used multiple AI tools including Claude, ChatGPT, and Gemini to assist with development. These tools helped me improve the structure of the portfolio website, add interactive features, and debug issues during development. They were used to generate ideas for JavaScript interactivity such as project search, category filtering, dark mode toggle, and contact form feedback. AI was also used to review the HTML and CSS structure to ensure the website remained organized and responsive.

## Benefits and Challenges
Using AI tools helped speed up the development process and provided useful suggestions for improving the user interface and interactivity of the website. It helped me understand how to implement features like DOM manipulation, event listeners, and localStorage. One challenge was that AI sometimes generated solutions that were more complex than necessary, so I had to simplify the code to match my understanding and the assignment requirements.

## Learning Outcomes
Through using Claude, ChatGPT, and Gemini, I learned more about JavaScript event handling, filtering content dynamically, improving user feedback, and structuring front-end projects properly. I also improved my understanding of how HTML, CSS, and JavaScript interact to build a responsive and interactive web page.

## Responsible Use and Modifications
All AI-generated suggestions were carefully reviewed before being used. I modified the generated code to match my own project structure and simplified parts that were unnecessary. I also tested all features manually to make sure they worked correctly and ensured that I understood the final implementation used in the project.
